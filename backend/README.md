# TODO-List Backend

##  Backend Initial Set Up
```
$ pwd
.../TODO-List/backend

// 1. Install project dependencies
$ npm install

// 2. Start the development server
$ npm run dev
```

## Run the Backend Locally
```
$ pwd
.../TODO-List/backend

// 1. Install project dependencies
$ npm install

// 2. Start the development server
$ npm run dev
```

## Backend Structure
```
|-- backend
    |-- config
        |-- db.js
    |-- controllers
        |-- taskController.js
        |-- userController.js
    |-- middleware
        |-- authorization.js
    |-- routes
        |-- taskRoutes.js
        |-- userRoutes.js
    |-- schemas
        |-- Task.js
        |-- User.js
    |-- .env
    |-- server.js
    |-- package.json
    |-- package-lock.json
```
